# Kubera Roadmap

> Zero is better than negative.

## v0.1 — Foundation ✦ current
- [x] Installer TUI (`curl | bash` one-liner)
- [x] Docker Compose setup (web, worker, db, redis)
- [x] Multi-provider AI (OpenRouter, Ollama, Anthropic, OpenAI, custom)
- [x] Port conflict detection and suggestions
- [x] Auto-generated secure config files
- [x] Update / uninstall subcommands

## v0.2 — Debt Module
- [ ] Loan tracker (name, principal, interest rate, EMI, start date)
- [ ] Multiple payoff strategies: avalanche, snowball, hybrid
- [ ] Monthly payoff timeline with end date projection
- [ ] Progress bar per loan, overall debt freedom date
- [ ] Milestone celebrations (50% paid, last EMI, debt free!)
- [ ] AI explains why it recommends a payoff order

## v0.3 — Dividend SIP Planner
- [ ] User defines monthly income target (e.g. ₹25,000/month)
- [ ] User defines monthly SIP contribution
- [ ] AI suggests 2–3 stocks per month with reasoning
- [ ] NSE/BSE stock lookup with live dividend yield data
- [ ] SIP tracking — invested vs target, expected annual dividend
- [ ] "On track / Off track" status with suggested adjustments

## v0.4 — Portfolio Rebalancing
- [ ] Upload holdings (CSV/Excel or manual entry)
- [ ] Live LTP via Yahoo Finance
- [ ] Sector allocation view with target vs actual
- [ ] Monthly rebalancing suggestions from AI
- [ ] Buy/Hold/Trim/Sell recommendations per holding
- [ ] Expected passive income from current portfolio

## v0.5 — Recurring Expense Tracker
- [ ] Log recurring expenses (subscriptions, insurance, rent, EMIs)
- [ ] Smart reminders before due dates
- [ ] Monthly cash flow: income vs commitments vs free cash
- [ ] AI flags "you can afford to increase SIP by ₹X this month"
- [ ] Visualise where money goes before investing

## v1.0 — The Full Journey
- [ ] Onboarding flow: where are you? (in debt / at zero / building)
- [ ] Financial freedom date calculator
- [ ] Net worth timeline (past + projected)
- [ ] Monthly AI check-in report (email or in-app)
- [ ] Goal achievements and streaks
- [ ] Mobile-friendly PWA

## Beyond v1.0
- Mutual fund SIP support (not just stocks)
- NPS / PPF / ELSS tracking (Indian retirement instruments)
- Tax-loss harvesting suggestions
- Multi-currency support
- Family/household mode
