## Error Handling

Standard Rails `rescue_from` in `ApplicationController`. JSON API errors return `{ error: message }` with appropriate HTTP status. Sidekiq job failures auto-retry (configurable). Use `Rails.logger` for warnings/errors. No custom error classes unless necessary.
